Outlined text input with an uppercase eyebrow label. Moss focus border, rust error state.

```jsx
<OutlinedField label="Phone number" placeholder="(555) 012-3456" value={v} onChange={setV} />
<OutlinedField label="Password" type="password" error="Must be 8+ characters" />
```

56px tall, 16px radius. Pass `leadingIcon` for a flag/icon inside the field.
