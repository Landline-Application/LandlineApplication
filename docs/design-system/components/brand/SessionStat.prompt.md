A labeled metric tile — big Fraunces value over an uppercase label.

```jsx
<div style={{ display:'flex', gap:'var(--space-lg)' }}>
  <SessionStat label="Time Remaining" value="42m 18s" tone="primary" style={{ flex:1 }} />
  <SessionStat label="Notifications" value={12} tone="secondary" style={{ flex:1 }} />
</div>
```

Use two-up under the rotary dial when a session is active.
