The brand's hero container — a vintage Rolodex index card with a clay tab and two punch holes.

```jsx
<RolodexCard title="Sign In">
  <OutlinedField label="Phone" placeholder="(555) 012-3456" />
  <Button label="Continue" fullWidth />
</RolodexCard>
```

Reserve for branded moments (auth, welcome). The tab text is always uppercase. For plain content surfaces use `Card` instead.
